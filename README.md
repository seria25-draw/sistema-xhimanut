# sistema-xhimanut

Sistema de gestión operativa y contable para el Restaurante-Mirador El Xhimanut, 
ubicado en Gigante, Huila.

---

## Descripción

Sistema web multi-rol que digitaliza y centraliza el flujo de pedidos, 
facturación y administración del restaurante, permitiendo a meseros, 
cocina/bar, caja y administradores operar de forma coordinada y en tiempo real.

---

## Actores del sistema

| Rol | Funcionalidad principal |
|---|---|
| Mesero | Registra pedidos y solicita cuentas |
| Cocina / Bar | Recibe y despacha pedidos por zona |
| Cajero | Genera facturas y realiza cierre de caja |
| Administrador | Gestiona usuarios, reportes y dashboard |

1. Clona el repositorio
```bash

```
2. Abre el archivo `index.html` en tu navegador
3. Inicia sesión con las credenciales según el rol
